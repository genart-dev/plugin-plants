## Key Architecture Notes for Grammar Authoring

### Inflorescence Types (L-system patterns)
- **Raceme**: Linear axis, flowers on pedicels along stem → `F[+K]F[-K]F[+K]...`
- **Cyme**: Branching where terminal bud flowers first → deterministic, then sympodial
- **Umbel**: All pedicels from one point → `F[+FK][-FK][/FK][\FK]...`
- **Panicle**: Branched raceme → compound L-system with sub-racemes
- **Corymb**: Like raceme but lower flowers have longer pedicels → flat top
- **Spike**: Sessile flowers directly on stem → `F[K]F[K]F[K]...`

### Phyllotaxis Notes
- **Sunflower**: divergence = 137.508° (golden angle), c = 4.0, count = 1000+
- **Dandelion clock**: 3D spherical phyllotaxis, pappus = radial line bundle per seed
- **Zinnia**: tighter spiral, divergence ≈ 137.5°, fewer organs
